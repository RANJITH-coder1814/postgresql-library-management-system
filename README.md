# 📚 Library Management System (PostgreSQL)
Beginner PostgreSQL project to manage books, members, and book issue records using SQL relationships and constraints.

## 📌 Project Description
A beginner SQL project that manages books, members, and issued books in a library.

## 🛠 Technologies
- PostgreSQL
- SQL

## 🔥 Features
- Add members
- Add books
- Issue books
- View issued books
- Count issued books per member
- Check available copies

## 🧠 Concepts Covered
- PRIMARY KEY
- FOREIGN KEY
- CHECK constraint
- JOIN
- GROUP BY
- Aggregate Functions
